# 🚀 Pterodactyl Astronaut Theme

Tema kustom untuk Pterodactyl panel dengan nuansa luar angkasa biru tua, efek astronot, musik player, dan foto profil personal.

## ✨ Fitur
- 🌌 Background biru tua dengan bintang berkelap-kelip
- 👨‍🚀 Efek astronot terbang, UFO, planet, dan nebula
- 🎵 Musik player Spotify di pojok kanan
- 📸 Foto profil owner di pojok
- 🖼️ Logo bulat di halaman login
- 💬 (Coming soon) Fitur chat integrasi
